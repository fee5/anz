# ms-ui
ms ui

### develop
- `npm run dev:web` to develop frontend
- `npm run dev:app` to develop server

### build
- `npm run build:web` to build frontend code
- `npm run build:app` to build server code

### run
`node build/main.js` then open *http://localhost:8885*
